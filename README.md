# 🚀 Alfariz Programming - Company Profile

Official website for **PT. Alfariz Teknologi Nusantara**, a software development house specializing in Web, Mobile, and Desktop solutions.

## 🌐 Live Demo
You can visit the live website here: 
👉 **https://alfariz-programming.vercel.app**

## ✨ Features
* **Modern Hero Section:** Professional introduction to the company.
* **Service Cards:** Highlights for Web, Mobile, and Desktop development.
* **WhatsApp Style Testimonials:** Creative and trustworthy client feedback.
* **Interactive FAQ:** Smooth accordion for common customer inquiries.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Direct WhatsApp Integration:** Quick access for potential clients to consult.

## 🛠️ Tech Stack
This project was built using modern web technologies:
* **React 19** (JavaScript + React Compiler)
* **Vite** (Next Generation Frontend Tooling)
* **React Bootstrap** (For styling and responsive components)
* **React Icons** (Professional SVG icons)
* **Vercel** (Cloud deployment)

## 📦 Installation & Setup
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/ydhlfrz/alfariz-programming.git](https://github.com/ydhlfrz/alfariz-programming.git)

2. Navigate to the project folder:
   ```bash
   cd alfariz-programming

3. Install dependencies:
   ```bash
   npm install

4. Start the development server:
   ```bash
   npm run dev
